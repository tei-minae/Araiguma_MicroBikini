🐾 指の貫通対策（アルファマスクの使用方法）

本製品のケモノハンドは、構造上「素体の手足」と重なるため、指や足が貫通して見えます。
この問題は、付属のアルファマスクを使用することで解消できます。


■ 方法概要

素体の手足部分を透明化することで、ケモノハンドのみを表示させます。


■ 手順

  1. アバターの「Body」マテリアルを選択します
  2. シェーダーを lilToon に設定します
  3. Rendering Mode を Cutout に変更します
  4. 「メインカラー」タブを開きます
  5. 「アルファマスク」のプルダウンを「None」から「置き換え」に変更します
  6. 表示されたスロットに、付属のマスク画像（HandMask.png）を設定します

■ 設定後の状態

  • 素体の手足が透明になります
  • ケモノハンド（および足）のみが表示されます
  • 貫通が解消されます

⚠️
他のアバターで使用する場合
本製品に付属しているマスクは対応アバター専用です。他のアバターで使用する場合は、以下のようにマスク画像を作成してください。
  • 手足の部分を 黒 に塗りつぶす
  • それ以外の部分を 白 に塗りつぶす
作成したマスク画像を、上記手順と同様にアルファマスクへ設定してください。

■ 注意点

  • lilToon以外のシェーダーでは使用できません
  • UV配置が異なるアバターでは正しく動作しない場合があります

■ 補足

付属のマスク適用済みマテリアルを使用することで、設定を省略することも可能です。
既にマテリアルを改変している場合は、上記手順でマスクのみを追加してください。

🐾 Finger Clipping Prevention (Alpha Mask)

Due to the structure of the Kemono Hands in this product, they overlap with the base avatar's hands and feet, which can cause clipping.
This issue can be resolved by using the included alpha mask.


■ Overview

By making the base avatar's hands and feet transparent, only the Kemono Hands will be visible.


■ Procedure

  1. Select the avatar's "Body" material.
  2. Set the shader to lilToon.
  3. Change the Rendering Mode to Cutout.
  4. Open the "Main Color" tab.
  5. Change the "Alpha Mask" dropdown from "None" to "Replace".
  6. Assign the included mask image (HandMask.png) to the displayed slot.

■ Result

  • The base avatar's hands and feet become transparent.
  • Only the Kemono Hands (and feet) are visible.
  • Clipping is resolved.

⚠️
When Using with Other Avatars
The included mask is exclusively for the supported avatar. If you wish to use it with other avatars, please create a mask image as follows:
  • Fill the hand and feet areas with Black.
  • Fill all other areas with White.
Apply the created mask image to the alpha mask slot using the same procedure above.

■ Notes

  • This cannot be used with shaders other than lilToon.
  • It may not work correctly on avatars with different UV layouts.

■ Additional Info

You can skip the manual setup by using the included material with the mask already applied.
If you have already modified your material, please follow the steps above to add only the mask.

🐾 手指穿模对策（Alpha遮罩使用方法）

由于本产品的兽爪在结构上与“素体手脚”重叠，因此会出现手指或脚部穿模的现象。
该问题可以通过使用附带的Alpha遮罩（Alpha Mask）来解决。


■ 方法概要

通过将素体的手脚部分透明化，使其仅显示兽爪部分。


■ 步骤

  1. 选择模型素体的“Body”材质（Material）
  2. 将着色器（Shader）设置为 lilToon
  3. 将渲染模式（Rendering Mode）更改为 Cutout
  4. 展开 “主颜色”（Main Color） 选项卡
  5. 将 “Alpha遮罩”（Alpha Mask） 的下拉菜单从“None”更改为 “替换”（Replace）
  6. 将附带的遮罩图片（HandMask.png）拖入显示的槽位中

■ 设置后的效果

  • 素体的手脚将变透明
  • 仅显示兽爪（和兽脚）
  • 穿模问题得到解决

⚠️
在其他模型上使用时
本产品附带的遮罩仅适用于指定适配模型。若要在其他模型上使用，请按以下方式自行制作遮罩图片：
  • 将手脚区域涂成 黑色
  • 其余部分涂成 白色
制作完成后，同样按照上述步骤将该图片设置为Alpha遮罩即可。

■ 注意事项

  • 无法在 lilToon 以外的着色器上使用
  • 如果模型的UV布局不同，可能无法正常工作

■ 补充说明

您也可以直接使用附带的“已应用遮罩材质”来跳过此设置。
如果您已经修改了材质，请按照上述步骤仅添加遮罩图片。

🐾 손가락 관통 방지 대책 (알파 마스크 사용법)

본 제품의 짐승 앞발(케모노 핸드)은 구조상 '소체의 손발'과 겹치기 때문에 손가락이나 발이 뚫고 나오는 관통 현상이 발생합니다.
이 문제는 동봉된 알파 마스크(Alpha Mask)를 사용하여 해결할 수 있습니다.


■ 방법 개요

소체의 손발 부분을 투명하게 처리하여 짐승 앞발만 표시되도록 합니다.


■ 설정 순서

  1. 아바타의 'Body' 마테리얼을 선택합니다.
  2. 셰이더를 lilToon으로 설정합니다.
  3. Rendering Mode를 Cutout으로 변경합니다.
  4. '메인 컬러(Main Color)' 탭을 엽니다.
  5. '알파 마스크(Alpha Mask)' 풀다운 메뉴를 'None'에서 '대체(Replace)'로 변경합니다.
  6. 표시된 슬롯에 동봉된 마스크 이미지(HandMask.png)를 설정합니다.

■ 설정 후 상태

  • 소체의 손발이 투명해집니다.
  • 짐승 앞발(및 뒷발)만 표시됩니다.
  • 관통 현상이 해결됩니다.

⚠️
다른 아바타에서 사용할 경우
본 제품에 동봉된 마스크는 대응 아바타 전용입니다. 다른 아바타에서 사용할 경우 아래와 같이 마스크 이미지를 직접 제작해 주십시오.
  • 손발 부분을 검은색으로 칠한다
  • 그 외의 부분을 흰색으로 칠한다
제작한 마스크 이미지를 위와 같은 순서로 알파 마스크에 설정해 주십시오.

■ 주의사항

  • lilToon 이외의 셰이더에서는 사용할 수 없습니다.
  • UV 배치가 다른 아바타에서는 정상적으로 작동하지 않을 수 있습니다.

■ 참고

마스크가 이미 적용된 동봉 마테리얼을 사용하면 설정을 생략할 수 있습니다.
마테리얼을 이미 개변하신 경우, 위 순서대로 마스크만 추가해 주십시오.
